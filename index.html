Quick Start

Setup lokal dalam 3 langkah mudah

1

Clone Repository

git clone https://github.com/RizaAlraihany/student-assistant.git
cd student-assistant
2

Run Setup Script

chmod +x setup.sh
./setup.sh
3

Start Development Servers

# Terminal 1 - Backend
cd backend && php artisan serve

# Terminal 2 - Frontend
cd frontend && npm run dev
